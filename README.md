# Drilling Digital Twin with ROP Prediction

## Overview
This project aims to build a simplified digital twin of a drilling system and predict Rate of Penetration (ROP) using data-driven models.

## Features
- Input parameters: WOB, RPM.
- Random Forest regression model for ROP prediction
- Basic system-level representation of drilling process

## Data
The dataset used in this project is not publicly available and was provided for academic purposes.

## Current Status
Work in progress — model accuracy is being improved.

## Future Work
- Improve prediction accuracy
- Integrate physics-based modeling and simulation
- Expand system representation
