# Drilling Digital Twin with ROP Prediction

## Overview
This project aims to build a drilling rig simulation platform based on digital twin technology. It can be used to train engineers and graduate students as a supplement to study petroleum drilling engineering.

## Features
- The project has 3 modules: Drill Rig, Drill Operations and Drill Risks (currently being developed). 
- Each of the modules have both learning and testing modes.
- Basic system-level representation of drilling process.

## Data
The real dataset used in this project is not publicly available and similar synthetic dataset was provided for academic purposes.

## Models
3D models used in the project are not included in thsi repository due to licensing and distribution restrictions. Models are widely available through websites like Sketchfab or AI generators which you can use instead.

## Current Status
Work in progress — model accuracy is being improved.
